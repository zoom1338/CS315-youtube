# CS315-youtube
YouTube Data mining project for CPT_S 315

## Project Setup
1. Clone the Repository
```
git clone <repo-url>
cd <project-folder>
```

2. Create a virtual environemnt
```
python3 -m venv venv
source venv/bin/activate
```
*add venv into your .gitignore

3. Install packages
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
